# Purpose

This implementation uses the following :

* Database
    - H2

* Data access
    - Data JPA

* Network API
    - REST serving Hypermedia (manual)

* Developer side testing
    - Unit test framework
        - JUnit 5
    - Mutation testing
        - PIT
    - Assertions library
        - JUnit 5 + Hamcrest
    - Coverage reports
        - JaCoCo
    - Static code analysis
        - ?
    - Build acceptance testing
        - Newman
    - Integration testing
        - Spring Boot

* QA side testing
    - BDD automation
        - Cucumber
        - FitNesse
    - Build acceptance testing
        - Postman

* Database migrations
    - Liquibase

* Monitoring
    - Actuator

* Build tool
    - Maven
